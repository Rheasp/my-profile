#Programming goals
1. Learn the basics of programming, including variables, functions, loops, and conditionals.

2. Practice writing clean, organized, and readable code.

3. Build small projects to improve problem-solving skills and gain hands-on experience.

4. Learn how to use Git and GitHub to track changes, manage projects, and collaborate with others.

5. Improve debugging skills by learning how to find, understand, and fix errors in code.
